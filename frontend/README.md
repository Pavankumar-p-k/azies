# Project Aegis Frontend

React + Tailwind security-terminal dashboard.

## Quick Start

```bash
npm install
npm run dev
```

Set variables in the repository root `.env` file before running.

## Deploy on Vercel

1. Import the repo in Vercel.
2. Set project root to `frontend`.
3. Add env vars:
   - `VITE_API_BASE_URL`
   - `VITE_WS_URL`
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
4. Deploy.
